The Basics

tmux (terminal multiplexer) lets you create multiple terminal sessions, split your screen, and keep sessions alive even when you disconnect.

The Prefix Key

Almost all tmux commands start with the prefix key:

Ctrl + b

Press and release the prefix, then press the next key.

Ctrl + b then r
Reload tmux config
Shows "Config reloaded!" message

Sessions

A session is a workspace. Create one per project.

Terminal
$ tmux new -s myproject
# Creates a new session named "myproject"
$ tmux ls
# Lists all sessions
$ tmux attach -t myproject
# Attaches to an existing session
Ctrl + b then d
Detach from session
Session keeps running in background!
Ctrl + b then s
List all sessions
Interactive session picker
Ctrl + b then $
Rename current session

Windows (Tabs)

Windows are like browser tabs. Each window has its own set of panes.

0: editor
1: server
2: git

The status bar shows your windows

Ctrl + b then c
Create new window
Ctrl + b then n
Next window
Ctrl + b then p
Previous window
Ctrl + b then 0-9
Jump to window by number
Ctrl + b then ,
Rename current window

Panes

Split your window into multiple panes. This is where tmux shines!

Editor
Tests
Server
Ctrl + b then |
Split pane vertically
Creates pane to the right
Ctrl + b then -
Split pane horizontally
Creates pane below
Ctrl + b then m
Toggle zoom (fullscreen pane)
Great for focusing on one pane temporarily
Ctrl + d
Close current pane
Or just type "exit"

Vim-Style Navigation

This config uses vim-style keys to move between panes. No prefix needed!

Ctrl + k ↑ Up
Ctrl + h ← Left
Navigate panes
Ctrl + l → Right
Ctrl + j ↓ Down

Works with Neovim too!

This config detects when you're in Neovim and passes the keys through, so you can navigate seamlessly between Neovim splits and tmux panes with the same keys.

Resizing Panes

Use vim keys with the prefix to resize panes.

Resize me!
Ctrl + b then h
Resize left (5 cells)
Ctrl + b then j
Resize down (5 cells)
Ctrl + b then k
Resize up (5 cells)
Ctrl + b then l
Resize right (5 cells)

Repeatable!

After pressing the prefix, you can tap h/j/k/l multiple times to keep resizing without pressing the prefix again.

Copy Mode (Vim-Style)

Scroll through history and copy text using vim keybindings.

Ctrl + b then [
Enter copy mode
q or Esc
Exit copy mode

In Copy Mode

j / k
Move down / up
Ctrl + d / Ctrl + u
Page down / up
/
Search forward
?
Search backward
Space
Start selection
Enter
Copy selection and exit

Real Workflow Example

Here's how you might use tmux for a typical development session:

1

Start a project session

tmux new -s myapp
2

Create your layout

Prefix + | to split for code

Prefix + - to split for tests

3

Navigate with vim keys

Ctrl + h/j/k/l to move between panes

4

Create windows for different contexts

Prefix + c for new window

Prefix + , to name it

5

Detach when done

Prefix + d to detach

Session keeps running!

Practice Exercises

Open a terminal and try these:

Exercise 1: Basic Session

  1. Create a session: tmux new -s practice
  2. Detach: Prefix + d
  3. List sessions: tmux ls
  4. Reattach: tmux attach -t practice

Exercise 2: Pane Splits

  1. Split vertically: Prefix + |
  2. Split horizontally: Prefix + -
  3. Navigate all panes with Ctrl + h/j/k/l
  4. Zoom a pane: Prefix + m
  5. Unzoom: Prefix + m again

Exercise 3: Windows

  1. Create 3 windows: Prefix + c (x3)
  2. Name them: Prefix + ,
  3. Jump between: Prefix + 0/1/2

Exercise 4: Resize

  1. Create a split: Prefix + |
  2. Resize: Prefix + h (repeat to keep resizing)
  3. Try all directions: h/j/k/l
